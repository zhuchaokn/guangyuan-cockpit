## 4. 已知限制

| 项目 | 说明 | 影响 |
|------|------|------|
| 地图为 ECharts SVG 渲染 | 未接入高德/百度在线地图 API | 无路网/卫星底图，仅显示行政区划边界 |
| 数据为前端 Mock | 所有数值为硬编码假数据 | 每次刷新随机部分会变化 |
| 分辨率适配 | 基准 1920×1080 | 低于 1280px 宽度时侧面板可能拥挤 |
| 字体 CDN | Google Fonts | 需网络连接加载字体，离线使用需本地化 |
