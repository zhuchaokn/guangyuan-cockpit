# 广元市综合管控驾驶舱 — 需求规格说明书

**项目名称：** 广元市综合管控驾驶舱  
**文档版本：** v1.0  
**编写日期：** 2026-03-13  

**目标：** 构建全屏数据可视化大屏驾驶舱，用于广元市交通综合管控，涵盖 7 大模块 41 项功能。

**技术方案：** React 19 + Vite 8 + ECharts 6 + GeoJSON 行政区划地图，前端纯静态实现，数据使用 Mock 假数据。

---

## 全局设计规范

### 布局结构
- **分辨率基准:** 1920×1080（16:9），使用 rem + vw/vh 自适应缩放
- **顶部导航栏:** 高度 70px，包含系统标题「广元市综合管控驾驶舱」+ 7个模块 Tab 切换 + 实时时钟 + 天气
- **中间区域:** 高德地图全屏铺底
- **左侧面板:** 宽度 ~420px，根据当前模块显示不同数据卡片，可滚动
- **右侧面板:** 宽度 ~420px，根据当前模块显示不同数据卡片，可滚动
- **底部状态栏:** 高度 40px，显示系统运行状态、数据更新时间

### 视觉风格
- **主色:** 深蓝黑底 (#0a1628)，科技蓝边框和高亮 (#00d4ff, #0ea5e9)
- **辅色:** 告警红 (#ef4444)、预警橙 (#f59e0b)、安全绿 (#22c55e)、信息紫 (#8b5cf6)
- **卡片样式:** 半透明深色背景、蓝色发光边框、标题带装饰线
- **图表风格:** 深色背景、发光色彩、渐变填充
